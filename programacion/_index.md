# Programación y Desarrollo

Servicios y herramientas dedicadas al desarrollo de software, CI/CD y gestión de código fuente.

## Servicios Planificados

- [Gitea (Mirror de GitHub)](gitea.md): Servidor de git ligero configurado para mantener espejo bidireccional con GitHub.
- [Gitea Actions](gitea-actions.md): Runner (`act_runner`) para CI/CD en Gitea, compatible con la sintaxis de GitHub Actions.
- [Jenkins](jenkins.md): Servidor de automatización para pipelines de CI/CD.
- [VS Code Server](vscode-server.md): Entorno de desarrollo remoto accesible vía navegador.
